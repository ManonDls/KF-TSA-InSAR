Introduction 
************

This program implements time series analysis for Interferometric Synthetic Aperture Radar (InSAR) 
with a data assimilation approach. The algorithm is based on a Kalman smoother. 

See Dalaison & Jolivet,  *A Kalman Filter Time Series Analysis method for InSAR*,
Journal of Geophysical Research - Solid Earth, doi: 10.1029/2019JB019150 (2020)
